# Big Data - Tarea 4 MongoDB

## Caso de uso
Sistema de ventas online utilizando MongoDB.

## Descripción
Este proyecto implementa una base de datos NoSQL para gestionar información de clientes, productos, pedidos y pagos en una plataforma de comercio electrónico.

## Tecnologías utilizadas
- MongoDB
- MongoDB Compass
- JavaScript
- GitHub

## Funcionalidades implementadas
- Creación de base de datos
- Inserción de documentos
- Consultas CRUD
- Filtros
- Operador $lookup
- Consultas de agregación

## Colecciones
- customers
- products
- orders
- payments
